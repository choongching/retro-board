Design System

JomRetro

The visual language behind the multiplayer retrospective board. Cool blue-tinted neutrals, editorial restraint, and a single reusable component vocabulary. Every specimen below renders from the live styles.css, so this page is always true to production.

Type · Inter Tight Mono · Chivo Mono Theme · Cool blue Tokens · --color-* namespace

Cool blue is the single brand accent, reserved for primary actions and focus. Danger is the only other intent color, for destructive actions.

Components · Menu items

Dropdown rows

Full-width, left-aligned rows for popover and dropdown menus. Distinct from buttons: tighter radius, no border, .danger for destructive rows.
